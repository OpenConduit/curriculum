# Data Engineering Bootcamp: Zero to Hero

## Setup
1. Install [Docker Desktop](https://www.docker.com/products/docker-desktop/).
2. Clone this repository.
3. Copy the environment file:

```bash
cp .env.example .env
```

4. Start the lab.
```bash
docker-compose up -d
```

## How to Log In
1. Open your browser to: http://localhost:8888
2. Enter the password: dataengineer (or whatever is in your .env file).
3. Open notebooks/00_Start_Here.ipynb.
